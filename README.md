# todoApp-back
# to start project

1. Change Database Url at .env file
   eg. "mysql://root:[yourpassword_to_mySQL]@localhost:3306/todoApp"

2. Install Dependencie
   "pnpm install"

3. setup Prisma
   "pnpm prisma generate"
   "pnpm exec prisma migrate dev"

4. Run project with command
   "pnpm dev"
